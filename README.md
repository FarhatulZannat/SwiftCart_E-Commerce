1) What is the difference between null and undefined?
Ans- undefined মানে ভেরিয়েবল declare করা হয়েছে কিন্তু কোনো value দেওয়া হয়নি, আর null মানে ইচ্ছাকৃতভাবে value খালি রাখা হয়েছে।
2) What is the use of the map() function in JavaScript? How is it different from forEach()? 
Ans- map() প্রতিটি element থেকে নতুন একটি পরিবর্তিত array তৈরি করে, আর forEach() শুধু প্রতিটি element এর উপর কাজ চালায় কিন্তু কোনো নতুন array return করে না।
3) What is the difference between == and ===? 
Ans- == শুধু value মিলায় প্রয়োজনে type convert করে, আর === value ও type দুটোই একসাথে মিলায় এবং type convert করে না।
4) What is the significance of async/await in fetching API data? 
Ans- async/await API থেকে ডাটা আনার সময় Promise শেষ হওয়া পর্যন্ত অপেক্ষা করে কোডকে সহজ ও পরিষ্কারভাবে লিখতে সাহায্য করে।
5) Explain the concept of Scope in JavaScript (Global, Function, Block)
Ans- JavaScript এ Scope হলো ভেরিয়েবল কোথায় অ্যাক্সেস করা যাবে—Global scope সবার জন্য উন্মুক্ত, Function scope শুধু ওই ফাংশনের ভিতরে সীমিত, আর Block scope {} এর ভিতরে সীমাবদ্ধ।
